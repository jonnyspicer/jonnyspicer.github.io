In exciting news, this weekend I managed to fix my previously documented dual boot OS installation issues. Frustratingly, all I had to do was run the Ubuntu installer again with all my drives connected (which I'd already done once - or at least thought I had), and this time there were no bootloader issues. I guess the classic developer tactic of try-the-exact-same-thing-again- and-hope-for-a-different-result actually worked this time.
